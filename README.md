## Purpose of the Project
This repository has been established exclusively to host the comprehensive research files, datasets, and documentation for an upcoming multi-disciplinary academic monograph. The primary objective is to bridge the gap between digital cyber-warfare analysis and physical geospatial intelligence.

## Project Overview
Search-of-Stuxnet is an advanced, multi-disciplinary monographic research repository dedicated to the comprehensive deconstruction of the Stuxnet and Duqu cyber-physical warfare architectures. Moving beyond conventional, code-only analysis, this project bridges the gap between digital exploitation and tangible physical geography through a data-driven approach.

This repository hosts a multi-spectral and geospatial intelligence (GEOINT) framework utilizing Synthetic Aperture Radar (SAR) (including Gamma 0 dB and Linear Gamma 0 processing), SAR urban extraction models, and RGB ratio coloring to map, measure, and analyze critical infrastructure topographies, logistics lifelines, subterranean facility volumes, and peripheral defense networks. Concurrently, it consolidates deep kernel-level analysis regarding the Tilded platform, investigating old Windows NT internals, driver injection vectors, and industrial control system (ICS/SCADA) rootkit anomalies.
